# Global COVID-19 Impact Dashboard

![Global COVID-19 Impact Dashboard](Goodness_Festus_Week4_COVID19_Dashboard_Screenshot.jpeg)

## Project Overview

This project analyzes the historical global impact of COVID-19 using confirmed cases and deaths data from the Johns Hopkins CSSE COVID-19 dataset.

The dashboard was built in Microsoft Power BI to track global confirmed cases, total deaths, death rate, affected countries, country-level comparisons, and case trends over time.

## Objective

The objective of this project is to communicate COVID-19 insights clearly through interactive dashboards, KPI analysis, visual reporting, and data storytelling.

## Dataset Used

Dataset: Johns Hopkins CSSE COVID-19 Dataset

Files used:

* `time_series_covid19_confirmed_global.csv`
* `time_series_covid19_deaths_global.csv`

Period covered: January 2020 to March 2023

## Tools Used

* Microsoft Power BI
* Power Query
* DAX
* GitHub

## Key KPIs

* Total Confirmed Cases: 677M
* Total Deaths: 7M
* Death Rate: 1.02%
* Affected Countries: 201
* Latest Report Date: March 9, 2023

## Dashboard Features

* KPI cards
* Global confirmed cases trend
* Top 10 countries by confirmed cases
* Top 10 countries by deaths
* Global spread map
* Country summary table
* Year and country filters
* Key insights panel

## Key Insights

* Global confirmed cases reached 677M by March 2023.
* The United States recorded the highest confirmed cases and deaths.
* The global death rate was 1.02%.
* Countries such as Brazil, India, France, and Germany were among the most affected.
* The trend chart shows major growth in confirmed cases around 2021–2022.

## Challenges Faced

* Transforming the dataset from wide format to long format.
* Merging confirmed cases and deaths data correctly.
* Fixing Power BI aggregation issues with cumulative COVID-19 data.
* Designing a clean dashboard that remained readable on a dark theme.

## Recommendations

* Decision-makers should monitor early case growth trends to support faster public-health responses.
* Countries with high death counts or high death rates should receive stronger healthcare support.
* Dashboards should be used to communicate public-health data clearly to stakeholders.
